# RAG Editorial

> ИИ-помощник для решения алгоритмических задач на основе базы знаний.

Веб-приложение для решения олимпиадных и алгоритмических задач с помощью **Retrieval-Augmented Generation (RAG)**. Пользователь описывает задачу своими словами, система ищет похожие задачи с разборами в векторной базе, собирает контекст и генерирует обоснованный ответ через LLM — опираясь на реальные решения, а не на догадки модели.

Для студентов и участников олимпиад, которым нужна подсказка с объяснением, подкреплённая разборами похожих задач.

## ✨ Возможности

- **Ответы на основе RAG** — модель рассуждает, опираясь на реальные задачи и разборы.
- **Семантический поиск** по базе знаний.
- **Асинхронная обработка** — статус запроса обновляется в реальном времени.
- **Пополняемая база знаний** — встроенные парсеры собирают задачи с e-olymp и AtCoder.
- **JWT-аутентификация** с ролью администратора.

## 🛠️ Технологии

Python 3.12, FastAPI, FastStream (Kafka), SQLAlchemy, PostgreSQL, Qdrant, Yandex Cloud LLM, LangChain. Фронтенд — Express на чистом JavaScript.

## 🚀 Запуск

**Требования:** Python 3.12+, Poetry, Docker Compose, Node.js.

```bash
make install        # установка зависимостей

# создайте env.json в корне проекта (см. ниже)

make infra-up       # Postgres, Kafka, Qdrant
make migrate        # миграции БД

# каждый сервис в своём терминале:
make api            # FastAPI на http://localhost:8000
make streams        # обработчики конвейера
make front          # фронтенд на http://localhost:3000
```

Пример `env.json`:

```json
{
  "YANDEX_CLOUD_API_KEY": "<ваш-api-ключ>",
  "YANDEX_CLOUD_FOLDER": "<id-каталога>",
  "YANDEX_CLOUD_MODEL": "qwen3-235b-a22b-fp8/latest",
  "KAFKA_BOOTSTRAP_SERVERS": "localhost:9094",
  "JWT_SECRET_KEY": "<надёжный-секрет>"
}
```

Откройте **http://localhost:3000**, зарегистрируйтесь и отправьте первый запрос. Документация API — на `http://localhost:8000/docs`.
